## Context

EvoClaw 桌面 sidecar 架构无公网 IP，企微只支持 webhook push，无法直接对接 → **飞书是当前唯一可直接对接的企业 IM**（支持 WebSocket 长连接）。

2026-04-21 结合两份参考实现调研，决策完整复刻而非 MVP：

## 验收标准

端到端（需用户飞书自建应用）：
1. 飞书开发者后台配置：启用机器人能力 + 长连接模式 + 权限 `im:message`/`im:message.group_msg`/`im:message.p2p_msg`
2. 单聊 "你好" → Agent 回复
3. 群聊 @机器人 "你好" → Agent 回复；未 @ 不响应
4. Slash `/model` `/cost` 生效
5. Markdown 回复飞书显示富文本（Post）
6. 图片/文件上传下载正常
7. 审批卡按钮点击 → 权限生效 → 卡片更新
8. 长回复 (>500 字符) 走 streaming card 逐字展示
9. 断网 30s 自动重连恢复
10. 飞书文档 @机器人评论 → agent 参与评论协作
11. `pnpm test` 全绿，覆盖率 ≥ 80%
