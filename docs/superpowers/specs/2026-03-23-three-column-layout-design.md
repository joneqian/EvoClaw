# 三栏布局改造设计

**日期**: 2026-03-23
**状态**: 已确认

## 背景

参考 EasyClaw (OpenClaw 中文版) 的布局设计，将 EvoClaw 从两栏布局改为三栏布局，提升对话切换效率和整体用户体验。

## 目标布局

```
┌──────┬────────────┬──────────────────────────┐
│ 图标  │  我的专家    │  [专家设置]               │
│ 导航  │  ─────────  │                          │
│ 54px │  主龙虾      │     Hi, 我是123          │
│      │  股票助手    │         123              │
│ 对话  │  ▶ 123     │                          │
│ 技能  │  跨境电商    │  ┌──────────────────┐    │
│ 专家  │             │  │ 任何问题，尽管吩咐~  │    │
│ 定时  │ + 创建专家   │  │                  │    │
│ 连接  │ ─────────  │  │ [📎][✨] [模型▾][↑]│    │
│ 安全  │ 会话历史     │  └──────────────────┘    │
│      │  1231      │                          │
└──────┴────────────┴──────────────────────────┘
```

## 三栏结构

### 第一栏：图标导航 (54px, 固定)

- 纯图标按钮，无文字
- 导航项：对话、技能商店、专家中心、定时任务、连接、安全中心
- 当前选中项高亮 (品牌色背景)
- 顶部显示品牌 logo
- 不可折叠，始终显示

### 第二栏：专家面板 (~240px, 始终可见)

**上半部分：我的专家**
- 标题 "我的专家"
- Agent 列表（头像 + 名称 + 副标题/ID）
- 选中的 Agent 背景高亮
- 底部 "+ 创建专家" 按钮

**分割线**

**下半部分：会话历史**
- 标题 "会话历史"
- 最近对话列表（Agent 头像 + 对话标题）
- 滚动加载

**仅在"对话"导航选中时显示第二栏内容为专家+会话历史。**
其他导航页面（技能/专家中心/定时/连接/安全）仍然在第二栏+第三栏合并的区域渲染，保持现有页面布局。

### 第三栏：主内容区 (剩余宽度)

**对话页**：
- 顶部：Agent 名称 + "专家设置" 快捷按钮
- 中部：消息列表
- 底部：输入区（附件按钮 + AI 增强 + 内联模型选择器 + 发送按钮）

**其他页面**：占满第二栏+第三栏的空间。

## 内联模型选择器

- 位于输入框右侧、发送按钮左侧
- 显示当前 Agent 绑定的模型名称 (如 "Qwen3.5 Plus")
- 下拉可切换到其他已配置模型
- 选择后临时覆盖本次对话的模型（不改 Agent 配置）

## 涉及文件

| 文件 | 变更 |

## 术语

- "专家" = Agent（EvoClaw 概念，不用"员工"）
- "会话历史" = 最近对话列表
